Build

CoAP.NET currently supports various versions of drafts of CoAP. Define one of the following compile symbols to build specific draft of CoAP library:

  • COAP03 -- draft-ietf-core-coap-03
  • COAP08 -- draft-ietf-core-coap-08
  • COAP12 -- draft-ietf-core-coap-12
  • COAP13 -- draft-ietf-core-coap-13

Define COAPALL to have all supported drafts together. In this case, all supported drafts will be available in class Spec:

{% highlight csharp %}public static class Spec { public static readonly ISpec Draft03; public static readonly ISpec Draft08; public static readonly ISpec Draft12; public static readonly ISpec Draft13; }{% endhighlight %}
